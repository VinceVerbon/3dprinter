# 3dprinter

Working repo for Vince's 3D printing setup — printer config, slicer profiles, calibration data, prints, troubleshooting notes, and any helper scripts that grow up around the workflow.

## Layout

```
.
├── docs/                  # documentation (procedures, references, notes)
│   ├── releaselog.md      # short bilingual index of tagged releases
│   └── release-notes/     # narrative per-version release notes
├── CHANGELOG.md           # Keep-a-Changelog (developer-facing)
├── VERSION                # SemVer source of truth
└── README.md              # this file
```

## Versioning

SemVer. Pre-1.0, treat `MINOR` like post-1.0 `MAJOR`. The release procedure is
documented in `CLAUDE.md` ("Change & release trail").
