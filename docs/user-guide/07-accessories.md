# Accessories

The **Accessories** page tracks everything that *isn't* filament but keeps the
printer running — nozzles, hotends, build plates, AMS parts, fans, belts,
lubricants, glues, desiccants, tools, and cleaning supplies.

Open it from the **Accessories** entry in the top navigation.

## What you can record

Each accessory has:

| Field | Notes |
|-------|-------|
| **Brand** | The maker. |
| **Name** | What it is, e.g. *0.4 mm hardened nozzle*. |
| **Category** | One of: nozzle, hotend, build plate, AMS, fan, belt, lubricant, glue, desiccant, tool, cleaning, other. |
| **Sub-category** | Optional finer grouping. |
| **SKU** | Optional supplier code. |
| **In stock** | How many you have. |
| **Rating** | Your 1–5 star score. |
| **Notes** | Anything else. |

## Adding, editing, removing

- **Add** — click **Add accessory**, fill in the form, and **Save**.
- **Edit** — click the pencil on a card, change what you need, **Save**.
- **Remove** — click the remove button on a card.

## Filtering by category

The dropdown at the top filters the list to a single category, with a count next
to each. Choose **All categories** to see everything. The page heading shows the
total number of accessories.

## Where the catalog fits in

Haspel ships a read-only **catalog** of official P2S replacement parts and common
consumables. You don't add those here directly — instead you pick from the
catalog when building a [shopping list](08-shopping-list.md). Once you've actually bought and
received a part, add it here as an accessory to track that you own it.
